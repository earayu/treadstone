Everything an agent needs.
Nothing it doesn't.

Built for the specific workflow of AI agents — create on demand, run in isolation, hand off to a human when needed.

Isolated by default

Each sandbox runs in its own container. No bleed between agents, users, or tasks. Safe to run untrusted code.

// integrate

Three ways in.

CLI for exploration, Python SDK for automation, REST API for integration. All returning the same consistent JSON.

bash 
# 1. Authenticate (or set TREADSTONE_API_KEY in env)
$ treadstone auth login --email agent@example.com --password ••••••••
✓ Logged in as agent@example.com

# 2. See available templates
$ treadstone --json templates list
{"items": [{"name": "aio-sandbox-tiny", "cpu": "0.25", "memory": "512Mi"}, ...]}

# 3. Create a sandbox — read the ID from JSON output
$ treadstone --json sandboxes create --name agent-demo
{"id": "sb_3kx9m2p", "status": "running", "urls": {"proxy": "https://sb_3kx9m2p.proxy.treadstone-ai.dev"}}

# 4. Hand the browser off to a human
$ treadstone --json sandboxes web enable sb_3kx9m2p
{"open_link": "https://sb_3kx9m2p.web.treadstone-ai.dev?token=...", "expires_at": "2026-03-30T18:00:00Z"}
python — pip install treadstone-sdk 
from treadstone_sdk import Treadstone

client = Treadstone(api_key="ts_live_xxxx")

# List available templates
templates = client.templates.list()
print(templates[0].name)      # "aio-sandbox-tiny"

# Create a sandbox
sandbox = client.sandboxes.create(
    name="agent-demo",
    template="aio-sandbox-tiny",
)
print(sandbox.id)            # "sb_3kx9m2p"
print(sandbox.urls.proxy)    # "https://sb_3kx9m2p.proxy.treadstone-ai.dev"

# Generate a browser hand-off URL for a human
session = sandbox.web.enable()
print(session.open_link)     # "https://sb_3kx9m2p.web.treadstone-ai.dev?token=..."
print(session.expires_at)   # "2026-03-30T18:00:00Z"

# Lifecycle management
sandbox.stop()
sandbox.start()
sandbox.delete()

# Create a sandbox

// workflow

Designed for agent workflows.

Four steps from task to hand-off. Every step is a single command — or a single API call.

01 — AUTH
Authenticate once

Set an API key in your environment. All subsequent commands are fully non-interactive.

export TREADSTONE_API_KEY=ts_...
02 — CREATE
Create a sandbox

Pick a template by size. Receive a sandbox ID and proxy URL immediately in JSON.

sandboxes create --name task
03 — RUN
Execute your task

Run code, browse the web, or use tools inside the isolated container via the proxy URL.

sb_xxx.proxy.treadstone-ai.dev
04 — HAND OFF
Give a human the wheel

Generate a secure, time-limited browser session URL and share it. One command.

sandboxes web enable sb_xxx
