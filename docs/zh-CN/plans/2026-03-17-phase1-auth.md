# Phase 1：认证系统 实施计划

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 实现用户注册、登录、JWT 鉴权，为所有后续 API 提供认证基础。

**Architecture:** 基于 JWT 的无状态认证，密码用 bcrypt 哈希，token 在 header 中传递。SQLAlchemy async model + FastAPI Depends 注入。

**Tech Stack:** FastAPI, SQLAlchemy async, python-jose (JWT), passlib + bcrypt, Pydantic v2

**Step 2: 实现 User model**

```python
# treadstone/models/user.py
import uuid
from datetime import datetime

from sqlalchemy import String, DateTime, func
from sqlalchemy.orm import Mapped, mapped_column

from treadstone.core.database import Base


class User(Base):
    __tablename__ = "users"

    id: Mapped[uuid.UUID] = mapped_column(primary_key=True, default=uuid.uuid4)
    email: Mapped[str] = mapped_column(String(255), unique=True, index=True, nullable=False)
    password_hash: Mapped[str] = mapped_column(String(255), nullable=False)
    created_at: Mapped[datetime] = mapped_column(DateTime(timezone=True), server_default=func.now())
```

**Step 3: 生成 Alembic 迁移**

```bash
uv run alembic revision --autogenerate -m "add users table"
uv run alembic upgrade head
```

**Step 4: Commit**

```bash

git commit -m "feat: user model with alembic migration"
```
