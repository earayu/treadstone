# Treadstone Brand Assets

This directory stores the current Treadstone logo source files and export assets.

Files:

- `logo-symbol.svg`: Primary transparent-background symbol for light surfaces.
- `logo-symbol-inverse.svg`: Transparent-background inverse symbol for dark surfaces.
- `app-icon.svg`: Square app icon master on a white background.
- `app-icon-on-dark.svg`: Reversed square app icon master on a black background.
- `logo-lockup-horizontal.svg`: Horizontal lockup with the symbol and `Treadstone` wordmark.
- `logo-lockup-horizontal.png`: Exported PNG preview of the horizontal lockup.
- `app-icon-1024.png`: Exported 1024x1024 app icon.
- `app-icon-512.png`: Exported 512x512 app icon.
- `app-icon-256.png`: Exported 256x256 app icon.
- `app-icon-on-dark-1024.png`: Exported reversed 1024x1024 app icon.

Notes:

- The canonical source shape is the SVG artwork in this directory.
- Exploratory iterations remain under `.codex-artifacts/logo-explorations/`.
