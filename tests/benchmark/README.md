# Benchmark (load tests)

This directory holds an **internal** harness for **burst sandbox creation** and polling via the `treadstone` CLI. It is **not** part of the default test suite and is **not** run by CI.

**Procedures, commands, artifact formats, guardrails, and troubleshooting:** see the agent skill **[`.agents/skills/benchmark/SKILL.md`](../../.agents/skills/benchmark/SKILL.md)** (also listed in **[`AGENTS.md`](../../AGENTS.md)**).
